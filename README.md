# Kivoti

Kivoti ist eine Offline-First Kalorien-Tracking-App, die als Lernprojekt im Rahmen meiner Ausbildung zum Anwendungsentwickler entwickelt wird.

Ziel des Projekts ist es, professionelle Softwareentwicklung von der Planung bis zur Umsetzung zu erlernen. Dabei stehen Architektur, Datenmodellierung, Git-Workflows, Flutter, SQLite und API-Integrationen im Fokus.

## Ziel

Eine einfache, schnelle und benutzerfreundliche Alternative zu bestehenden Kalorien-Tracking-Apps.

Der Schwerpunkt liegt auf einer unkomplizierten Erfassung von Lebensmitteln und einer langfristig erweiterbaren Architektur.

## MVP Features

* Lebensmittelverwaltung
* Kalorientracking
* Gewichtsverlauf
* Wassertracking
* Tagesübersicht
* Kalenderansicht
* Wochenstatistiken

## Geplante Features
